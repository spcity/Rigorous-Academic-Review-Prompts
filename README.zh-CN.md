
# 严格学术审稿 Prompts

一套面向学术论文、理论工作和研究方法审查的严格型 Prompt 模板集合。

这个仓库适合希望获得比普通“论文审稿 Prompt”更严格、更技术化、更结构化评审能力的用户。仓库中包含两个核心 Prompt：

- 全面学术审稿 Prompt
- 研究方法专用审查 Prompt

同时支持针对以下方向进行更严格的分流审查：

- 通用学术论文
- 机器学习理论 / 优化
- 纯数学 / 理论计算机

## 项目简介

本仓库提供一组用于严格审稿的 Prompt 模板，重点用于：

- 全文审稿
- 方法专审
- 数学公式与理论正确性检查
- 证明、推导、假设与边界条件审查
- 创新性、合理性与细节完整性评估
- 输出结构化、可执行的修改建议

这些 Prompt 适用于：

- 论文初稿自审
- 技术报告评估
- 理论型论文检查
- 机器学习理论 / 优化论文审查
- 纯数学 / 理论计算机论文审查
- 研究方法或算法设计审查

## 包含内容

### 1. 全面审稿 Prompt comprehensive-review-prompt.md
这是一个面向整篇学术稿件的严格型审稿 Prompt，用于评估稿件的：

- 正确性
- 严谨性
- 假设条件
- 创新性
- 证据支撑
- 形式化精确性
- 理论可靠性
- 评估质量

该 Prompt 支持三路自动分流：

- 通用学术审稿
- 机器学习理论 / 优化审稿
- 纯数学 / 理论计算机审稿

### 2. 方法专用 Prompt method-review-prompt.md
这是一个只聚焦“研究方法本身”的方法审查 Prompt。

它重点评估：

- 方法正确性
- 方法合理性
- 方法创新性
- 细节完整性
- 假设清晰度
- 可复现性
- 内部一致性
- 方法相关证据是否充分

它也支持自动分流到不同的方法审查路径。
