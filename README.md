# Rigorous Academic Review Prompts

A curated collection of rigorous prompts for reviewing academic manuscripts, theoretical work, and research methods.

This repository is designed for users who want stricter, more technically grounded, and more structured review prompts than generic "paper review" prompts. It includes both a comprehensive academic review prompt and a method-focused review prompt, with specialized routing for general academic work, machine learning theory / optimization, and pure mathematics / theoretical computer science.

## Overview

This repository contains prompt templates for:

- Full academic manuscript review
- Method-only review
- Stronger scrutiny of mathematical correctness
- Stronger scrutiny of theoretical rigor
- Stronger scrutiny of algorithmic validity
- Structured review outputs with actionable revision suggestions

These prompts are suitable for reviewing:

- Research paper drafts
- Technical reports
- Theory-heavy manuscripts
- ML theory and optimization papers
- Pure mathematics and theoretical CS papers
- Method sections or standalone research method descriptions

## Included Prompts

### 1. Comprehensive Review Prompt
A full, publication-style review prompt for evaluating a manuscript's:

- correctness
- rigor
- assumptions
- novelty
- evidence
- formal precision
- theoretical soundness
- evaluation quality

It supports automatic routing across three review tracks:

- General Academic
- ML Theory / Optimization
- Pure Math / TCS

### 2. Method Review Prompt
A focused prompt for evaluating only the research method itself.

It is designed to assess:

- method correctness
- method reasonableness
- method innovation
- detail completeness
- assumption clarity
- reproducibility
- internal coherence
- method-specific supporting evidence

It also supports automatic routing across multiple method-review tracks.
